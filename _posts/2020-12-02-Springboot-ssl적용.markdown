> HTTP & HTTPS  

**HTTP**   
* 인터넷 상에서 정보를 주고 받기 위한 프로토콜  
* 클라이언트와 서버 사이에 이루어지는 요청/응답 프로토콜
* 암호화되지 않은 방법으로 데이터를 전송한다.

**HTTPS**   
* 보안이 강화된 HTTP
* Hypertext Transfer Protocol Over Secure Socket Layer 의 약자
* 모든 HTTP 요청과 응답 데이터는 네트워크로 보내지기 전에 암호화를 한다.
